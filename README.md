# Batonlib：一个高性能协程网络库

## Introduction

本项目为C++编写的协程库，并在此基础上封装了网络部分，对系统调用进行了封装，用户不需要关心阻塞以及协程的调度。另外还实现了异步日志，记录服务器运行状态。
